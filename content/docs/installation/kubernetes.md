Kubernetes support for Dragonfly will be made available in Dragonfly v0.2.0 via Helm charts + Kustomize.

Helm charts will be available in the Dragonfly Helm repository, Kustomize overlays will be available in the Dragonfly GitHub repository.

# Prerequisites
- Kubernetes version >= 1.21
- Helm version >= 3.0.0

Further details are coming soon (Dragonfly v0.2.0)
